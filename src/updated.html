Yet Another Speed Dial has been updated to version 3!


Whats New?

  • dial format setting: you can now choose between wide or square dials:
    note the default is slightly wider than in v2, this is following the open graph standard so more images fit the dial automatically.
  • dial size setting
  • add image from url
  • custom background color for individual dials
  • open in background tab added to context menu
  • improved background color algorithm
  • improved image loading algorithm
  • improved import and export:
    bookmarks are now included in the export. yasd also supports importing from fvd dial and speed dial 2 (experimental)
  • search dials with ctrl+f
    page must be focused, if the address bar is focused you get the default find window
  • lots of other changes to support manifest v3 - this was a major change required by google. if you notice any issues please let me know on GitHub. thank you for your support!

YASD will always be 100% free and open source -- it is entirely supported by a community of volunteers which is why we can offer it without ads, a premium version, or selling user data. If you find it useful, please consider leaving a review for Chrome or Firefox ‚ù§
